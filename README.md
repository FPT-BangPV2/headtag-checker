# headtag-checker
MetaTag Auditor - Chrome &amp; Edge Extension Phát hiện ngay lập tức các thẻ &lt;title>, &lt;meta description>, &lt;h1>-&lt;h6>, &lt;link rel="canonical">, &lt;meta robots>, Open Graph, Twitter Cards bị trùng lặp hoặc thiếu trong phần &lt;head> và trên trang. Công cụ không thể thiếu của SEO specialist, developer và content editor.
